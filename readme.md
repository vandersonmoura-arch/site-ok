# Projeto Landing Page Admin - Vanderson Alves

Este projeto é um estudo prático de como gerenciar estados e persistência de dados no navegador usando **Vite** e **LocalStorage**.

## 🚀 Como rodar
1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. No terminal, na pasta do projeto, execute:
   ```bash
   npm install
   npm run dev
